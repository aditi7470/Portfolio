# Aditi Suhane - Portfolio (Node/Express/EJS/MySQL)

## Quick start

1. Copy `.env.example` to `.env` and set your DB credentials.
2. Create the database and table using `create_db.sql`.
3. Install dependencies:
   ```
   npm install
   ```
4. Run:
   ```
   node server.js
   ```
5. Open http://localhost:3000

## Notes
- Place your images in `public/images/` (filenames used in the template).
- If you prefer SQLite instead of MySQL, let me know and I will provide a SQLite version.
